# soal1mudah
Penjelasan singkat:

Program meminta input berat, jarak, panjang, lebar, tinggi.

Menghitung volume (panjang × lebar × tinggi).

Jika jarak ≤ 10 km → Rp 4250/kg, jika lebih → Rp 6000/kg.

Jika volume > 100 cm³ → tambah biaya Rp 50.000.
